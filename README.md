🚀 Project Showcase: Power BI Dashboard Integrated with AI Chatbot

I’m excited to share a full-stack analytics project that combines Business Intelligence (Power BI) with Artificial Intelligence (LLMs) into a single interactive web application.

📌 Project Title
 Power BI Dashboard with AI Chatbot Integration 

🔍 Project Overview
Traditional dashboards are powerful for visualization, but users often struggle to go beyond charts to understand insights deeply.
To solve this, I built an interactive web-based system where:
A Power BI dashboard is embedded directly into a web application
An AI-powered chatbot works alongside the dashboard
Users can analyze data visually and ask real-time questions through chat
This creates a unified experience that combines data visualization + intelligent interaction.

🧠 Key Features
✔ Embedded Power BI dashboard using iframe integration
 ✔ AI chatbot powered by a Large Language Model 
 ✔ Side-by-side interactive layout (Dashboard + Chat Interface)
 ✔ Flask-based backend with REST API architecture
 ✔ Modular and scalable full-stack project structure

🛠️ Technology Stack
Backend:
Python
Flask (Web Framework)
Groq API

Frontend:
HTML
CSS
JavaScript 

Business Intelligence:
Power BI Service
Embedded Reports via iframe

🏗️ System Architecture (High-Level Flow)
Power BI report is published to Power BI Service
The report is embedded into the web application
Flask backend serves the UI and handles chatbot requests
User queries are sent from frontend to backend via JSON
Backend processes queries using Groq LLM
AI-generated responses are returned in real time
This architecture cleanly separates data visualization and AI reasoning, while delivering a seamless user experience.

🎯 Use Case Applications
This system can be applied in multiple domains, including:
Healthcare analytics
Business performance monitoring
Executive reporting dashboards
Decision support systems
Example user interactions:
View KPI dashboards in Power BI
Ask contextual questions like “Why did this metric drop?”
Receive instant AI-generated explanations and insights
